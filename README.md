## ✨ Features

- **Lossless Compression & Decompression** using the powerful [LZ-String](https://pieroxy.net/blog/pages/lz-string/index.html) algorithm  
- Compress plain text or upload files for instant compression  
- Decompress base64 compressed strings back to original text  
- Responsive, minimalistic, and clean interface focused on usability  
- All processing occurs **locally in your browser** — your data stays private 🔒  
- Copy buttons with instant feedback for easy clipboard operations 📋  
- Mobile-friendly design ensures smooth experience on all devices 📱💻

## ⚙️ How to Use

### Compress Text or File

1. Enter or paste your text into the "Compress" text area **or** select a file.  
2. Click **Compress**.  
3. Your compressed output (base64 string) appears below — copy or save it.  

### Decompress Base64 String

1. Paste your compressed base64 string into the "Decompress" text area.  
2. Click **Decompress**.  
3. The original text output appears below for easy access.
